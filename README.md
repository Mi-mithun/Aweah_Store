# AwrahStore – Premium E‑Commerce Frontend

AwrahStore is a modern, fully responsive e-commerce product listing interface designed to showcase premium products. Built with vanilla JavaScript and Bootstrap 5, this project demonstrates core frontend development skills including dynamic product rendering, search functionality, admin controls, and persistent data storage.

Users can browse products displayed as interactive cards, each showing product image, category, name, short description, and price. A real-time search bar filters products by name, category, or description without page reload. The admin panel allows adding new products with name, price, category, image URL, and description — all fields have fallback defaults if left empty. Products can also be removed with a smooth delete animation.

Clicking "View Details" opens a Bootstrap modal with full product information including title, price, description, category, and dynamic tags. The modal also includes "Buy Now" and "Add to Cart" action buttons with toast notifications for user feedback.

All product data is stored in a JavaScript array and rendered dynamically using the `renderProducts()` function. LocalStorage support saves the product catalog so data persists across page reloads. The project also features a sticky navbar with active section highlighting, scroll-based animations using Intersection Observer, and a fully responsive layout that works seamlessly on mobile, tablet, and desktop devices.

## Key Features
- Dynamic product card rendering with JavaScript
- Real-time search filter (name, category, description)
- Admin panel: add / remove products with validation
- Bootstrap modal for detailed product view
- LocalStorage for data persistence
- Toast notifications for user actions
- Smooth scroll and fade-in animations
- Fully responsive (mobile to desktop)
- Sticky navbar with active section highlighting

## Technologies Used
- HTML5
- CSS3 (custom variables, animations, grid/flex)
- JavaScript (ES6+: map, filter, find, push, LocalStorage)
- Bootstrap 5 (grid, modal, toast, navbar)
- Font Awesome / Bootstrap Icons
- Google Fonts (Syne, DM Sans)
